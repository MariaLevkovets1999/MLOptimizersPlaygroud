Optimization Algorithms Playground

In this notebook, you can explore how different optimization algorithms behave when searching for a local minimum of a one-dimensional function.

The implemented algorithms include:

Gradient Descent
Momentum
Nesterov Momentum

Interactive widgets allow you to adjust key parameters such as:

the number of iterations
the learning rate
momentum

By tuning these parameters, you can observe how each algorithm moves across the function landscape.
